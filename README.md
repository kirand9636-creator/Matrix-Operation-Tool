# Matrix Operations Tool

A simple Python project that performs basic matrix operations using the NumPy library. This project allows users to enter two matrices and perform different mathematical operations through a menu-driven interface.

## Features

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Transpose of Matrix A
- Transpose of Matrix B
- Determinant of Matrix A
- Determinant of Matrix B
- Input validation for matrix dimensions
- Menu-driven command-line interface

## Technologies Used

- Python 3
- NumPy

- Python 3.x

Install NumPy using:

```bash
pip install numpy
```

## How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code or any Python IDE.
3. Install NumPy if it is not already installed.
4. Run the program:

```bash
python matrix_operations.py
```

## How It Works

1. Enter the dimensions of Matrix A.
2. Enter the values of Matrix A.
3. Enter the dimensions of Matrix B.
4. Enter the values of Matrix B.
5. Choose an operation from the menu.
6. View the result.
7. Repeat operations or exit the program.

## Example Operations

- Addition
- Subtraction
- Multiplication
- Transpose
- Determinant

## Input Rules

- For addition and subtraction, both matrices must have the same dimensions.
- For multiplication, the number of columns in Matrix A must equal the number of rows in Matrix B.
- Determinant can only be calculated for square matrices.
